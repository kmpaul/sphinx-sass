Installation¶

Sphinx-sass is compatible with Python versions >=3.6 and Sphinx >=1.8, and should work anywhere that Python and libsass-python can be installed.

Note

Sphinx-sass depends on libsass-python, which in turn depends on libsass. This latter dependency requires a recent C++ compiler (see the README for the libsass project for details).

Install from GitHub¶

To install sphinx-sass directly from the GitHub repository using pip:

$ pip install git+https://github.com/mwibrow/sphinx-sass.git

Alternatively, clone the repository and run the set-up script manually:

$ git clone https://github.com/mwibrow/sphinx-sass.git
$ cd sphinx-sass
$ python setup.py install
